Инвариантное моделирование

    Принцип системности (S-принцип)

    Любой объект, процесс или явление рассматривается как система.

    Гиперкомплексная динамическая система (ГДС)

    Система есть совокупность системных инвариант:

    S=\sum_{i=1\dots5}S_i=S_1+S_2+S_3+S_4+S_5

    • S_i — системная инварианта
    • S_1 — гиперкомплексность
    • S_2 — динамичность
    • S_3 — структурность
    • S_4 — эмержентность (целостность)
    • S_5 — иерархичность

    Эмержентность (целостность) S_4

    Такое свойство системы в целом, которого нет и не может быть у элементов системы, рассматриваемых отдельно.

    Иерархия

    Характеристика вложенности системы.

    Соподчиненность — частный случай иерархии.

    Порядок практической реализации и теоретического рассмотрения системных инвариант

    Каждая последующая системная инварианта может быть реализована тогда и только тогда, когда полностью реализованы все предыдущие системные инварианты.

    S\to S_1S_2S_3S_4S_5

    Системообразующая среда S_0

    Исходные данные, из которых строится система

    Уровень системной организации C

    Под уровнем системной организации будем подразумевать количество системных инвариант в анализируемой либо конструируемой системе.

    Как следует из определения ГДС, может быть от одного до пяти уровней системной организации. Или в процентном выражении, принимая за 100% наличие в системе пяти инвариант, можно говорить об интервале от 0 до 100% (в относительных единицах — от 0 до 1) в процессе системной реализации.

    • S_1 — 20%
    • S_2 — 40%
    • S_3 — 60%
    • S_4 — 80%
    • S_5 — 100%

    Требование адекватности модели исследуемому объекту

    С_М \ge C_О

    • С_М — уровень системной организации модели или метода
    • С_О — уровень системной организации исследуемого объекта

    Гиперкомплексный граф

    Элемент обозначается окружностью произвольного радиуса.

    Взаимодействие между элементами обозначается направленной стрелкой.

    Целостность отображается окружностью, либо другой замкнутой кривой, содержащей внутри себя системные элементы.

    Иерархия проявляет себя после графического отображения целостности: внешняя характеристика системы A (верхний иерархический уровень) и внутренние элементы A_1 и A_2 (нижние иерархические уровни).

    Обычный граф может быть получен из гиперкомплексного путем свертки — устранением из системы элементов низшего иерархического уровня (ликвидацией характеристики вложенности):

    Гиперкомплексная матрица

    Для матрицы без иерархии:

    • Порядок матрицы равен числу элементов системы.
    • Элементы располагаются на главной диагонали матрицы. Номер элемента и порядковый номер матричной клетки должны быть идентичны: A_1 \to M_{1.1}
    • Межсистемное взаимодействие помещается именно в те клетки, номер которых соответствует номеру взаимодействия: Y_{1.2} \to M_{1.2}

    Внутреннее устройство элемента, содержащего элементы низшего иерархического уровня, представляется в виде вложенной матрицы с соответствующим заполнением ее клеток.

    Порядок гиперкомплексной матрицы

    Величина, описываемая выражением в виде многомерной дроби, имеющей вид:

    \Pi = \alpha,\beta,\gamma,\dots

    • \Pi — порядок матрицы;
    • \alpha — максимальное число элементов системы на высшем иерархическом уровне;
    • \beta — максимальное число элементов на следующем (на одну ступеньку ниже) иерархическом уровне;
    • \gamma — максимальное число элементов на следующем, более низком иерархическом уровне, и так до самого нижнего иерархического уровня.

    Принцип системной реализации (R-принцип)

    Каждая ГДС находится в состоянии процесса системной реализации.

    По вертикали отложен уровень системной реализации A, по горизонтали — время t.

    Фазы системного развития:

    1. Предсистемное состояние, состояние системообразующей среды
    2. Фаза развития
    3. Фаза стационарности, состояние целостности
    4. Фаза распада
    5. Постсистемное состояние, состояние системообразующей среды

    Основной закон ГДС

    Каждая система стремится к реализации функции гиперкомплексного гиратора.

    S = S'+S''

    Разомкнутая составляющая:
    \lim_{t \to \infty }S'(t)=0

    Замкнутая составляющая:
    \lim_{t \to \infty} S''(t) = S'' \mid Y_{mn} = -Y_{nm}

    \lim_{t \to \infty} S(t) = S'' \mid Y_{mn} = -Y_{nm}

    Каждая система S может быть разложена на две составляющие: источник (дивергенцию) и циркулятор (ротор):

    S = \text{Div}(S) + \text{Rot}(S)

    Соотношение гиперкомплексных неопределенностей

    \prod_{i=1 \dots \infty}\Delta_i=\Delta_1\Delta_2\dots\Delta_n\dots=1=\text{const}\ne f(t)

    \Delta_i — гиперкомплексная неопределенность

    В простейшем случае:

    \Delta_1\uarr\cdot\Delta_2\darr=1=\text{const}\ne f(t)

    Гиперкомплексный спектр

    Применяется для отображения количественных характеристик и взаимного соотношения гиперкомплексных неопределенностей друг с другом.

    Пирамида иерархического развития

    Габариты = высота × ширина:

    Г = h \times a

    Для того, чтобы достичь высоты, превышающей собственную максимальную высоту пирамиды, необходимо расширить ее базис: увеличить число исходных элементов нижнего иерархического уровня.

    Иерархический принцип системной реализации

    По вертикали отложен уровень системной реализации A, по горизонтали — время t.

    Римскими цифрами обозначены зоны иерархического развития (от нижней первой до высшей четвертой).

    Буквой Z обозначена зона кризисного развития.

    Единицы измерения по вертикали изменяются при переходе с одного иерархического уровня на другой.

    Фазы (состояния) системного развития:

    1. Предсистемное состояние, состояние системообразующей среды
    2. Фаза развития
    3. Фаза стационарности, состояние целостности
    4. Фаза распада
    5. Постсистемное состояние, состояние системообразующей среды
    6. Кризисное состояние, зона неопределенности
    7. Движение по иерархии

    Зона кризисного развития (Z)

    В пределах зоны кризисного развития не выполняются никакие системные законы.

    Существует только три физически реализуемых варианта выхода из зоны кризисного развития:

    1. Движение по иерархии
    2. Сохранение стационарного режима
    3. Разрушение системы

    Вероятность выхода из зоны кризисного развития в любом направлении одинаково неопределенна.

    Движение или процессы в зоне кризисного развития определяются, диктуются и реализуются ситуационной целесообразностью.

    Непосредственно взаимодействовать между собой могут только системы (или элементы) одинакового иерархического уровня.

    Принцип соответствия

    Если система S' взаимодействует с системой S'' по принципу соответствия, то порядок исходных и результирующей матриц одинаков.

    На основе принципа соответствия система может менять только свои количественные характеристики. Изменений по качеству не происходит. Выйти в движение по иерархии нельзя.

    Принцип дополнительности

    Если система S' взаимодействует с системой S'' по принципу дополнительности, то порядок результирующей матрицы будет равен системной сумме порядков исходных взаимодействующих матриц.

    Реализация межсистемного взаимодействия на основе принципа дополнительности является определением и необходимым условием выхода в иерархическое развитие и получения нового качества.

    Нуль-транспортировка

    S' \to S_0 \to S''

    Ограничения на применения Инвариантного моделирования

    Ограничение снизу: ИМ нецелесообразно применять в том случае, когда исследуемый объект одномерен по качеству. Для качественно однородных процессов разработано огромное число эффективных конкретных методов, предназначенных для решения задач именно такого качества. ИМ — это метод решения междисциплинарных задач.

    Ограничение сверху определяется этической нормой: для получения конкретного эффекта при использовании методологии ИМ необходимо, чтобы уровень системной организации духовных и этических норм пользователя был выше уровня системной организации применяемых им теоретических и практических инструментальных средств.

    Cтруктурная матрица Y(С)

    Матрица, полученная из системной матрицы путем замены величин всех имеющихся в ней элементов и взаимодействий символом единицы, а отсутствующих элементов и взаимодействий — символом нуля.

    Фантом

    Искусственно введенный, отсутствующий в реальности системный элемент, наличие которого обусловливается определением системы, способом и порядком ее развития.

    Ортогональное разложение системной матрицы

    Используется при количественном анализе системы на основе основного закона ГДС.

    Y(S) = Y' + Y''

    Симметрическая матрица (соответствует дивергентной части системы): Y' = \frac{Y + Y^T}{2}

    Кососимметрическая матрица (соответствует роторной части системы): Y'' = \frac{Y - Y^T}{2}

    • Y — исходная матрица
    • Y^T — транспонированная матрица

    Модель

    Целеопределенный эквивалент:

    M = (C;Э)

    Системная модель:

    M = (C;S)

    • M — модель
    • C — деятельностная цель
    • Э — модельный эквивалент
    • S — системный эквивалент моделируемого объекта

    Системная цель C

    Эквифинальное состояние системы
    • Собственная (внутренняя) цель C_0 — устойчивое стационарное состояние, достигнутое системой в процессе самоорганизации.
    • Внешние цели C_n — целевые характеристики внешней среды по отношению к выделенной исходной системе.

    Расхождение целей:
